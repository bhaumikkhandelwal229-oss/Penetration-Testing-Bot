# Penetration Testing Bot

## Overview

The **Penetration Testing Bot** is a simple cybersecurity automation tool developed in Python.
This project is designed to assist security researchers and ethical hackers in performing basic reconnaissance and vulnerability discovery tasks.

The bot automates common penetration testing activities such as scanning targets, identifying open ports, and gathering basic information about the system.

This project is created for **educational and ethical security testing purposes only**.

---

## Features

* Automated target scanning
* Open port detection
* Network reconnaissance
* Simple command line interface
* Lightweight and easy to use
* Beginner-friendly cybersecurity project

---

## Technologies Used

* Python
* Nmap
* Requests Library

---

## Installation

1. Clone the repository

```bash
git clone https://github.com/bhaumikkhandelwal229-oss/penetration-testing-bot.git
```

2. Navigate to the project directory

```bash
cd penetration-testing-bot
```

3. Install required dependencies

```bash
pip install python-nmap requests
```

4. Make sure **Nmap** is installed on your system.

Download Nmap:
https://nmap.org/download.html

---

## Usage

Run the script using:

```bash
python PT.py
```

Enter the target IP address when prompted and the bot will begin scanning.

Example:

```
Enter Target IP: 127.0.0.1
```

The tool will display open ports and basic information about the target.
